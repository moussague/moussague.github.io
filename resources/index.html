Resources

As a developer and researcher, I rely on a variety of tools to streamline my workflow, enhance my productivity. This page is a curated collection of my essential resources, categorized for easy navigation. I hope you find them as valuable as I do!

These resources are essential to my workflow, enabling me to learn, research, code, and collaborate effectively.

I hope this collection provides you with valuable tools and inspiration for your own journey. I’d love to hear about your favorite resources as well – feel free to share your recommendations!

Research#

  • Zotero: My go-to tool for organizing research papers and generating citations. It’s incredibly helpful for managing references and creating bibliographies.
  • TeX Live: A comprehensive TeX distribution that provides everything I need for writing and typesetting documents in LaTeX.

Development & Coding#

  • VSCodium: My preferred code editor and Integrated Development Environment (IDE). It’s open-source, highly customizable, and has a vast library of extensions that make coding more efficient and enjoyable.
  • Git: An essential version control system that allows me to track changes to my code, collaborate with others, and manage different versions of my projects. I use it for all my coding endeavors to keep my work organized and ensure that I can always revert to previous versions if needed.

Graphics & Design#

  • GIMP: A powerful open-source image editor that I use for a wide range of image manipulation tasks, from basic photo editing to creating graphics and illustrations.
  • Inkscape: A versatile vector graphics editor that I use for creating scalable graphics and illustrations. It’s particularly useful for designs that need to be resized without losing quality.

Others#

  • AnyDesk: A powerful remote desktop application that I use for accessing and controlling my computers from anywhere. It’s incredibly useful for remote work, collaboration, and providing technical support.
